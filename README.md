# Smart Study Planner

## Description
Smart Study Planner is a full-stack web application designed to help students plan, organize, and track their study schedules efficiently.

## Tech Stack
- Frontend: React
- Backend: Django, Django REST Framework
- Database: PostgreSQL
- Authentication: JWT (planned)

## Features
- User authentication
- Create and manage study schedules
- Subject-wise planning
- Progress tracking dashboard
